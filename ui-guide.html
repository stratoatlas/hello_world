Known issues

Technical debt in existing pages. These are not fixed yet — they are documented here to prevent repeating the pattern.

Typography debt
~35–40 unique type combinations in existing files

All pages were written with inline font-size/weight/letter-spacing. The 12-style system above is the target — existing files are not yet refactored to use class names.

Button contrast
entry-btn was color:var(--text3) + border:.13

Old ghost button was near-invisible. New spec: .c-btn-ghost uses border:.10 default → .28 hover, text3 → text. Still needs to be applied to all instances in index.html.

--line opacity
Old: rgba(255,255,255,.07) — invisible on some monitors

Raised to .10 in this guide. Needs to propagate to style.css and all inline uses in existing pages.

--text2 contrast
Old: #7a8699 — borderline on laptops

Raised to #8f9db2. Needs to propagate to style.css and all hardcoded color values.

Fixed in v14
TOC pill inconsistency (3 different styles)

About TOC and Research TOC now use the same .c-pill component spec.
